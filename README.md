# Eco Kilimani Roots (EcoKili) üå±

> Smart Waste Management for Kilimani. From Waste to Wonder.

EcoKili is a community-driven platform designed to transform waste management in the Kilimani area. By connecting residents with waste collectors and gamifying the recycling process, we aim to build a cleaner, sustainable, and more beautiful community.

## ‚ú® Features

- **Waste Pickup Requests**: Schedule pickups for various sorted waste types.
- **EcoPoints System**: Earn points for every pickup (10 points per waste type) and track your contribution.
- **Community Progress**: Visualize collective impact with metrics like installed PET RainTiles and active members.
- **Sorting Guide**: Educational resources on how to properly sort waste.
- **User Roles**: Support for Residents and Waste Collectors (via Firebase Auth).

## üöÄ Getting Started

### Prerequisites

Make sure you have Node.js installed on your machine.

### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2.  Navigate to the project directory:
    ```bash
    cd eco-kilimani-roots
    ```
3.  Install dependencies:
    ```bash
    npm install
    # or
    bun install
    ```

### Development

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:8080`.

### Build

To build the project for production:

```bash
npm run build
```
